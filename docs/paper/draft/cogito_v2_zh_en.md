#  Co-Evolving Cognitive States and Solutions for Stateful AI Research Agents
## 摘要
AI研究智能体近年来在自动化机器学习工程方面展现出强劲潜力，然而大多数现有系统在认知层面仍然是根本性无状态的：它们在候选代码方案上进行搜索，却不显式维护一个关于任务的、持续演化的理解。这一局限导致了浅层的试错、跨步骤的任务特异性知识积累薄弱，以及在具有挑战性的多步问题上探索效率低下。

为解决这一问题，我们提出Cogito，一个基于元认知状态演化动力学构建的有状态AI研究智能体框架。与将记忆视为被动存储或将反思视为一次性修复工具的做法不同，Cogito维护一个显式的结构化认知状态，该状态随时间持续积累任务理解、工作假设、已学习的失败模式、搜索偏好和自评估的置信度。在每个执行步骤之后，Cogito利用多维执行反馈——包括性能趋势、错误类型、新颖度和置信度信号——通过元认知反思来更新这一认知状态。更新后的状态随即被反馈回来，用以指导后续的算子选择和方案生成。以这种方式，研究不再是对解空间的盲目遍历，而成为一个闭环过程，其中积累的认知持续塑造着未来的搜索和精炼方向。

在AIRS-Bench的20个跨领域任务上，Cogito在30步预算下取得了0.522的平均归一化分数，优于AIRA-dojo在200步和10个随机种子条件下取得的0.402。在总体性能之外，我们还进行了因果状态干预实验，将演化后的认知状态替换为空白状态、来自无关任务的状态或冻结快照。这些干预表明，认知状态携带了任务特异性信息，并能因果性地改善下游决策，而非仅仅充当额外的提示词上下文。

## Abstract

AI research agents have recently shown strong promise in automating machine learning engineering, yet most existing systems remain fundamentally stateless at the cognitive level: they search over candidate code solutions without explicitly maintaining an evolving understanding of the task. This limitation leads to shallow trial-and-error, weak accumulation of task-specific knowledge across steps, and inefficient exploration on challenging multi-step problems. To address this, we introduce Cogito, a framework for stateful AI research agents built on a metacognitive state-evolution dynamics. Instead of treating memory as passive storage or reflection as a one-step repair tool, Cogito maintains an explicit structured cognitive state that accumulates task understanding, working hypotheses, learned failure patterns, search preferences, and self-assessed confidence over time. After each execution step, Cogito uses multi-dimensional execution feedback, including performance trends, error types, novelty, and confidence signals, to update this cognitive state through metacognitive reflection. The updated state is then fed back to guide subsequent operator selection and solution generation. In this way, research is no longer a blind traversal of the solution space, but a closed-loop process in which accumulated cognition continuously shapes future search and refinement. On AIRS-Bench across 20 cross-domain tasks, Cogito achieves an average normalized score of 0.522 with a 30-step budget, outperforming AIRA-dojo’s 0.402 obtained with 200 steps and 10 seeds. Beyond aggregate performance, we conduct causal state intervention experiments by replacing the evolved cognitive state with blank states, unrelated task states, or frozen snapshots. These interventions show that the cognitive state carries task-specific information and causally improves downstream decisions, rather than merely serving as additional prompt context.
